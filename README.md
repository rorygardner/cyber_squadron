# Virginia County-Level Mobility Shifts Analysis (2020–2022)

This repository contains the data processing, statistical analysis, and replication code for the paper evaluating county-level mobility shifts across Virginia using Google Community Mobility Reports.

## About the Project

This codebase is designed to reproduce the findings of our study comparing mobility trends between metropolitan and non-metropolitan counties in Virginia. The pipeline includes handling missing values via bidirectional linear interpolation, applying Principal Component Analysis (PCA) to extract a “mobility away from home” index, and running exact county-level permutation inference.
